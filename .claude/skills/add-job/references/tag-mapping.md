# Job Tag Mapping

Use these tags when creating jobs. Match tags based on the job title, description, required skills, and company context.

## Special Tags (Use Sparingly)
| Slug | Label | When to Use |
|------|-------|-------------|
| `hot` | HOT | High-priority/urgent positions - requires explicit user request |
| `top` | TOP | Featured/premium positions - requires explicit user request |
